# Dashboard de E-commerce com Dash

Este projeto apresenta um dashboard interativo desenvolvido com Dash e Plotly para análise de dados de produtos de e-commerce.

## Gráficos incluídos

- Histograma
- Dispersão
- Mapa de calor
- Barras
- Pizza
- Densidade
- Regressão

## Como executar

pip install -r requirements.txt  
python app.py

O dashboard estará disponível em:
http://127.0.0.1:8050
